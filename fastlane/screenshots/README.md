# Screenshots for App Store

## Required sizes for macOS

- **Mac**: 2880 Ã— 1800 pixels (required)

## File naming

Screenshots should be named with prefix numbers for ordering:

```
en-US/
  01_menubar.png
  02_dictation.png
  03_hotkeys.png
  04_history.png
  05_settings.png

ru/
  01_menubar.png
  02_dictation.png
  03_hotkeys.png
  04_history.png
  05_settings.png
```

## Recommended screenshots

1. **Menu bar** - App icon in menu bar with dropdown open
2. **Dictation** - Recording in progress with waveform
3. **Hotkeys** - Settings showing hotkey configuration
4. **History** - Transcription history view
5. **Settings** - Settings panel with API key field

## Tips

- Use clean desktop background
- Show actual transcription results
- Highlight key features
- Keep consistent style across all screenshots
