# macOS Screenshots Guide — Говорилка для Mac

## Ключевые факты о конверсии

- **90% пользователей** не листают дальше 3-го скриншота
- **7 секунд** — среднее время на странице приложения
- Текст на скриншотах **индексируется Apple** для поиска
- Оптимизированные скриншоты дают **+20-35%** к конверсии

---

## Структура: Value → Usage → Trust

| Скриншот | Цель | Что показываем |
|----------|------|----------------|
| 1 | **Внимание** | Живёт в меню-баре |
| 2-3 | **Интерес** | Диктовка и хоткеи |
| 4 | **Желание** | История записей |
| 5 | **Доверие** | Приватность |

---

## Required Sizes

| Display | Resolution | Required |
|---------|-----------|----------|
| Mac | 2880 x 1800 | **Обязательно** |
| Mac (alternative) | 1280 x 800 | Optional |

## Цветовая палитра

### Основной градиент фона:
```
#FFF5F8 → #FFE4EC (вертикальный, сверху вниз)
```

### Акцентный градиент (для карточек):
```
#FF69B4 → #FFB6C1 (диагональный)
```

### Цвет текста:
- На светлом фоне: `#5D4E6D`
- На тёмном/ярком: `#FFFFFF`

---

## Художественные эффекты

### Используем:
- ✅ **Мягкие тени** — black 4% opacity, blur 6-8px
- ✅ **Glassmorphism** — полупрозрачные карточки с размытием
- ✅ **Ambient glow** — свечение вокруг облачка-маскота
- ✅ Чистый минималистичный десктоп

### Избегаем:
- ❌ Жёсткие тени
- ❌ Захламлённый рабочий стол
- ❌ Перенасыщенные цвета

## Recording Screenshots

### Menu Bar Screenshot
1. Click menu bar icon to open popover
2. Use Cmd+Shift+4, then Space to capture window
3. Or use full screenshot and crop

### Floating Window
1. Start recording (triggers floating window)
2. Position over desired app
3. Screenshot the composition

### Auto-paste Demo
1. Open text editor
2. Complete a recording
3. Screenshot immediately after paste

## Инструменты для создания

### Рекомендации:

1. **Figma** — основной инструмент
   - Создать компоненты для переиспользования
   - Автолейаут для разных размеров

2. **Rotato** — для 3D mockups устройств

3. **Shots.so** — быстрое создание скриншотов с рамками

### Экспорт:
- PNG, без сжатия
- sRGB цветовое пространство
- Без альфа-канала
