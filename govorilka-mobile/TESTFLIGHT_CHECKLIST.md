# Чеклист тестирования Говорилка (TestFlight)

## Перед тестированием
- [ ] Установить TestFlight из App Store
- [ ] Принять приглашение на тестирование
- [ ] Установить Говорилка через TestFlight

---

## 1. Первый запуск

### Разрешения
- [ ] Появляется запрос на доступ к микрофону
- [ ] После разрешения приложение работает корректно
- [ ] При отказе — показывается понятное сообщение

### Онбординг
- [ ] API ключ Deepgram можно ввести в настройках
- [ ] Ключ сохраняется после перезапуска приложения

---

## 2. Основная функциональность

### Запись и транскрипция
- [ ] Нажатие на кнопку записи начинает запись
- [ ] Визуальная индикация записи (анимация, уровень звука)
- [ ] Речь распознаётся в реальном времени
- [ ] Interim текст (серый) появляется во время речи
- [ ] Final текст (чёрный) появляется после пауз
- [ ] Повторное нажатие останавливает запись
- [ ] Финальный текст корректно отображается

### Качество распознавания
- [ ] Русский язык распознаётся корректно
- [ ] Английский язык распознаётся (если поддерживается)
- [ ] Пунктуация добавляется автоматически
- [ ] Числа распознаются корректно

### Копирование
- [ ] Текст автоматически копируется в буфер обмена
- [ ] Можно вставить в другое приложение (Notes, Messages)

---

## 3. История записей

- [ ] Записи сохраняются в историю
- [ ] Отображается дата и время
- [ ] Отображается длительность записи
- [ ] Можно скопировать текст из истории
- [ ] Можно удалить запись
- [ ] История сохраняется после перезапуска

---

## 4. Pro режим (если есть)

- [ ] Можно сделать фото
- [ ] Фото отправляется вместе с аудио
- [ ] Контекст из фото улучшает распознавание

---

## 5. Настройки

- [ ] API ключ можно изменить
- [ ] Звуки можно включить/выключить
- [ ] Вибрация можно включить/выключить
- [ ] Очистка текста работает корректно
- [ ] Настройки сохраняются после перезапуска

---

## 6. Обработка ошибок

### Сеть
- [ ] Без интернета — понятное сообщение об ошибке
- [ ] После восстановления сети — работает

### API ключ
- [ ] Неверный ключ — понятное сообщение
- [ ] Пустой ключ — предложение ввести

### Микрофон
- [ ] Если микрофон занят — сообщение об ошибке
- [ ] Если разрешение отозвано — инструкция как включить

---

## 7. UI/UX

### Внешний вид
- [ ] Розовая тема отображается корректно
- [ ] Шрифты читаемые
- [ ] Кнопки достаточного размера для нажатия
- [ ] Анимации плавные

### Разные устройства
- [ ] iPhone SE (маленький экран)
- [ ] iPhone 14/15 (стандартный)
- [ ] iPhone Pro Max (большой экран)
- [ ] Ориентация: только портрет (или ландшафт?)

### Доступность
- [ ] VoiceOver работает (если поддерживается)
- [ ] Dynamic Type (крупный шрифт) не ломает UI

---

## 8. Производительность

- [ ] Приложение запускается быстро (< 3 сек)
- [ ] Нет лагов при записи
- [ ] Батарея не разряжается быстро
- [ ] Приложение не крашится

---

## 9. Фоновый режим

- [ ] Запись продолжается при сворачивании
- [ ] Или корректно останавливается с сохранением

---

## 10. Edge cases

- [ ] Очень длинная запись (5+ минут)
- [ ] Очень короткая запись (< 1 сек)
- [ ] Тишина во время записи
- [ ] Громкий фоновый шум
- [ ] Быстрая речь
- [ ] Несколько языков в одной записи
