## Notes for Reviewer

```
This app requires a Deepgram API key to function.

To test the app:
1. Create a free account at https://console.deepgram.com
2. Generate an API key (free tier includes $200 credits)
3. Enter the API key in the app's Settings

The app captures audio from the microphone, streams it to Deepgram's
speech-to-text API, and displays the transcription in real-time.

Key features:
- Real-time speech recognition
- Copy transcription to clipboard
- History of past transcriptions
- Pro mode with photo context (optional)

No backend server required - direct API connection.
```
