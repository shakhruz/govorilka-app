Говорилка — минималистичное приложение для преобразования речи в текст с использованием передового AI Deepgram Nova-2.

ПОЧЕМУ ГОВОРИЛКА?

• Бесплатный старт — $200 кредитов при регистрации в Deepgram (≈775 часов записи)
• Без подписки — платите только за использование ($0.0043/мин)
• Приватность — аудио не сохраняется, история только на устройстве
• Открытый код — полная прозрачность на GitHub

ВОЗМОЖНОСТИ

◉ Мгновенная транскрипция
Говорите — текст появляется в реальном времени. Технология WebSocket стриминга обеспечивает минимальную задержку.

◉ Умная очистка текста
Автоматическое удаление слов-паразитов ("ну", "как бы", "типа", "значит"). Включается одним тапом в настройках.

◉ История записей
Все транскрипции сохраняются локально. Быстрый поиск, копирование и удаление.

◉ Pro-режим
Делайте фото во время записи. Идеально для обратной связи, баг-репортов и заметок с контекстом. Экспорт в Google Drive.

◉ Звуки и вибрация
Тактильная обратная связь при записи. Настраиваемые звуковые уведомления.

ОТЛИЧНОЕ РАСПОЗНАВАНИЕ РУССКОГО ЯЗЫКА

Deepgram Nova-2 — одна из лучших моделей для русской речи. Точность выше, чем у большинства конкурентов.

КАК НАЧАТЬ

1. Скачайте приложение
2. Получите бесплатный API-ключ на deepgram.com
3. Вставьте ключ в настройках
4. Нажмите кнопку записи и говорите!
